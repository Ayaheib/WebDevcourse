==================================================
STEP 5 – Add Flex to make cards wrap
==================================================
Turns `.card-grid` into a horizontal Flex container that wraps its children.
Each card keeps a minimum width and auto-flows into new rows.

Code to copy (add to flex.css):
----------------------------------------
.card-grid {
  display: flex;
  flex-wrap: wrap;          /* allow cards to move onto new lines */
  gap: var(--gap);          /* even spacing between cards */
}

.card {
  flex: 1 1 240px;          /* grow and shrink, base width = 240px */
  min-width: 220px;
  padding: 16px;
}
----------------------------------------
- Preview:
Now the cards line up in a row on large screens and wrap gracefully on smaller screens.





==================================================
`flex-wrap` and `flex-basis`
==================================================
`flex-wrap: wrap` lets items flow onto new lines when space runs out.  
`flex-basis` (the third number in `flex: 1 1 240px`) sets a *preferred* starting width.  
Together they create responsive rows without media queries!

Try live:
1. Change `flex-basis` → `180px` → cards get narrower.  
2. Resize browser → see them wrap automatically.






==================================================
STEP 6 – Make cards fill full height of each row (optional)
==================================================
 What it does:
Aligns cards so all in a row share equal height.

Code to copy (add below previous rule):
----------------------------------------
.card-grid {
  align-items: stretch;     /* stretch cards to same height per row */
}
----------------------------------------

Preview:
All cards in the same row are equal height – neat and balanced.
